# DESIGN CORRECTIONS & PRODUCT STRUCTURE

## PRODUCT NAME

The platform name is:

# INTERNAL

This is an internal operating system for Mavin Records.

Do not use:

* Mavin Insights
* Mavin Analytics
* Mavin Dashboard

The logo should simply read:

INTERNAL

with a clean enterprise SaaS identity.

---

# DESIGN PHILOSOPHY

The design MUST closely follow Spotify for Artists.

The experience should feel:

* Data-dense
* Professional
* Enterprise
* Music industry focused
* Fast
* Clean
* Serious

Avoid:

* Purple-heavy UI
* Web3 aesthetics
* Neon effects
* Glassmorphism
* Floating gradients
* Abstract blobs
* Circular futuristic widgets

The dashboard should feel like software used daily by label executives.

---

# MOBILE RESPONSIVE EXPERIENCE

The application must be fully responsive.

Create designs for:

* Desktop
* Laptop
* Tablet
* Mobile

On mobile:

Use bottom navigation.

Tabs:

Home

Artists

Search

AI

Campaigns

Profile

Artist switching should become a horizontal scrolling roster.

Cards should stack vertically.

Tables should transform into mobile cards.

Charts should become swipeable.

The AI assistant should work like ChatGPT mobile.

Follow these visual rules:

Light gray background

8px-12px border radius

Subtle shadows

Large artist imagery

Minimal iconography

Generous spacing

Heavy typography for metrics

Clean tables

Dense analytics

No excessive colors

No decorative graphics

No oversized gradients

The focus should always be:

Artist Data
Streams
Audience
Campaigns
Insights

Not decoration.
